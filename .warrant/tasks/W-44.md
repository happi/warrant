## Intent

The warrant CLI mishandles backlog-backed projects: missing frontmatter keys can abort task listing under set -e, backlog status names are not mapped cleanly, create --help can create a task, and server ID counters can drift below local task maxima causing duplicate low-ID allocations. Make backlog mode robust and safe for real repos like hh.
