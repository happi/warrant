## Intent

The server has zero automated tests. The CI compiles but does not verify behavior. A regression in CAS or the hash chain would go unnoticed.

## Decision

Add EUnit tests for the core modules: status CAS logic, hash chain computation, lease expiry. Test at the module level, not through HTTP.
