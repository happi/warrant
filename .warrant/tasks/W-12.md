## Intent

The CLI still calls the server API for task list and task get. It should read from local files since the repo is the source of truth.

## Decision

`warrant task list` parses YAML frontmatter from `.warrant/tasks/*.md`. `warrant trace` combines the task file with `git log`. Server calls only for ID allocation, CAS, and leases.
