## Intent

Installing the CLI requires cloning the repo or downloading a tarball. Most developers expect `npm install -g` or `brew install`.

## Decision

Start with npm since the CLI is bash + jq (no compilation). Wrap in a thin npm package that copies scripts to the bin path. Homebrew tap later if there is demand.
