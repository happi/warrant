No end-to-end test proves that the CLI, server, and git conventions work together. A breaking change in the API contract between client and server would only be caught manually.

## Decision

A shell script that starts the server, creates a task via CLI, transitions status, links a commit, reads the trace, and verifies each step. Run in CI.
