# Changelog

## 0.3.0 (2026-03-19)

- Renamed to "Warrants" for marketplace publication
- Added activation for `backlog/config.yml` and `backlog/config.yaml` workspaces
- Added marketplace metadata (icon, repository, keywords, license)

## 0.2.0 (2026-03-19)

- Local-first task reading from `.warrant/tasks/*.md`
- Sidebar with Current Task and Task List views
- Hover provider for task IDs in code and commit messages
- Inline blame decorator showing task IDs per line
- Task action commands: start, review, done, link commit, create
- Trace view with commits, branches, and audit trail
- Status bar showing current task
- File watcher for live task updates
- Server mode for leases, CAS, and hash chain (optional)

## 0.1.0 (2026-03-18)

- Initial release
- Server-only mode with API client
- Basic task tree view
